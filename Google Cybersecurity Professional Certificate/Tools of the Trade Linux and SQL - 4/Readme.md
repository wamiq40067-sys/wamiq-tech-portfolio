# Tools of the Trade: Linux and SQL

This file contains my learning notes, key concepts, and understanding of Linux fundamentals, command-line operations, file systems, permissions, and SQL basics as part of the Google Cybersecurity Professional Certificate.

---

## Topic Overview

This course focused on two essential tools used in cybersecurity: **Linux** and **SQL**. It introduced how security professionals use Linux systems for system administration, file management, and security operations, along with how SQL is used to interact with and manage databases. The course builds foundational skills for working in real-world cybersecurity environments.

- Introduction to databases and SQL

## Key Topics Covered

### Linux Fundamentals
- Overview of Linux as an open-source operating system
- Importance of Linux in cybersecurity environments
- Linux distributions and their use cases
- Terminal and command-line interface usage

### Linux Command Line Operations
- Navigating directories (`cd`, `ls`, `pwd`)
- File operations (create, copy, move, delete)
- Viewing file contents and system information
- Using commands efficiently for automation and control

### Linux File System and Permissions
- Linux directory structure
- File ownership and user roles
- Read, write, and execute permissions
- Managing access control for security

### Process and System Management
- Viewing running processes
- Managing system resources
- Understanding background and foreground processes
- Basic system monitoring concepts

### Introduction to SQL
- What databases are and how they store data
- Role of SQL in cybersecurity and data analysis
- Structure of tables, rows, and columns

### SQL Queries and Data Retrieval
- Writing basic SELECT statements
- Filtering data using WHERE clauses
- Sorting and organizing query results
- Understanding conditions and operators in SQL

### SQL in Cybersecurity
- Using SQL to analyze logs and security data
- Identifying suspicious activity in databases
- Extracting relevant information for investigations
- Supporting incident response with structured queries
