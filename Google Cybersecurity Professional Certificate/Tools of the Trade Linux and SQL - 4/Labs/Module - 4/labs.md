# Module 4 Labs Overview — *Tools of the Trade: Linux and SQL*

Module 4 focuses on learning how to use SQL to access databases, retrieve information, filter records, and combine data from multiple tables. The labs provide practical experience with SQL queries in cybersecurity-related scenarios.

---

# Activity: Perform an SQL Query

This lab introduces the basics of SQL queries using `SELECT` and `FROM`. Learners practice retrieving data from database tables and understanding how information is organized in relational databases.

## Skills Practiced
- Writing basic SQL queries
- Using `SELECT` and `FROM`
- Retrieving table data
- Understanding database structure

---

# Activity: Filter a SQL Query

This lab focuses on filtering records using the `WHERE` clause and comparison operators. Learners narrow down query results to display only relevant information.

## Skills Practiced
- Using `WHERE`
- Applying filters
- Using comparison operators
- Retrieving targeted records

---

# Activity: Apply More Filters in SQL

This lab expands filtering techniques by working with numeric values, dates, and multiple conditions. Learners investigate login attempts and suspicious activity using SQL filters.

## Skills Practiced
- Filtering numeric data
- Filtering date and time data
- Combining multiple conditions
- Investigating security-related events

---

# Activity: Filter with AND, OR, and NOT

This lab introduces logical operators used to create advanced SQL filters. Learners practice combining conditions to retrieve more specific results.

## Skills Practiced
- Using `AND`
- Using `OR`
- Using `NOT`
- Creating advanced query filters
- Pattern matching with `LIKE`

---

# Portfolio Activity: Apply Filters to SQL Queries

This portfolio project simulates real cybersecurity analyst responsibilities. Learners investigate suspicious login attempts and identify employee devices requiring security updates using SQL queries.

## Skills Practiced
- Investigating login activity
- Filtering employee records
- Using SQL in cybersecurity scenarios
- Applying multiple SQL filters
- Documenting query results

---

# Activity: Complete a Join

This lab introduces SQL joins and demonstrates how to combine information from multiple related tables. Learners connect employee and machine data using shared columns.

## Skills Practiced
- Using `INNER JOIN`
- Combining related tables
- Matching records with keys
- Retrieving linked information
- Working with relational databases

---

# Overall Module 4 Outcomes

By completing the labs in Module 4, learners gain practical experience in:

- Writing SQL queries
- Filtering and sorting data
- Investigating cybersecurity events
- Using logical operators
- Working with relational databases
- Combining data with joins
- Applying SQL to real-world security tasks

These labs help build foundational SQL skills used in cybersecurity analysis and security operations.
