# Module 2 Labs Overview

## Activity: Install Software in a Linux Distribution
In this lab, software was installed using Linux package management tools. The process involved updating repositories and installing required applications.

This helped understand how software is managed securely and efficiently in different Linux distributions.

---

## Activity: Examine Input and Output in the Shell
In this lab, standard input and output in the Linux shell were explored using basic commands. It demonstrated how data flows between commands and the terminal.

Pipes and redirection were used to observe how commands process and transfer information.
