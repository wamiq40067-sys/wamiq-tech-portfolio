# Module 3 Labs Overview

## Filesystem Navigation Lab
Practiced navigating the Linux filesystem using commands like `pwd`, `ls`, and `cd`. Explored directories and worked with absolute and relative file paths.

## File and Directory Management Lab
Created, moved, copied, and deleted files and directories using commands such as `mkdir`, `touch`, `mv`, `cp`, and `rm`.

## File Content Viewing Lab
Used commands like `cat`, `head`, `tail`, and `less` to read and inspect file contents efficiently from the terminal.

## Filtering and Searching Lab
Learned how to filter information using `grep`, pipes (`|`), and `find` to search for files, keywords, and recently modified content.

## Permissions Management Lab
Explored Linux file permissions and ownership using `ls -l` and modified permissions with the `chmod` command.

## User and Group Administration Lab
Practiced creating, modifying, locking, and deleting user accounts with commands like `useradd`, `usermod`, and `userdel`.

## File Ownership Lab
Used the `chown` command to change file and directory ownership for users and groups in Linux.

## Nano Text Editor Lab
Created and edited files using the `nano` text editor and practiced saving and exiting files through keyboard shortcuts.

## Standard Output Redirection Lab
Worked with output redirection operators `>` and `>>` to overwrite or append content into files using commands like `echo`.

## Linux Help Commands Lab
Used support commands such as `man`, `apropos`, and `whatis` to access Linux documentation and command descriptions.
