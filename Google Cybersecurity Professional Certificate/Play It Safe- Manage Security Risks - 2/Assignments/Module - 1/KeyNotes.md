# Security Domains

## 🔐 Key Notes

The CISSP framework is a globally recognized cybersecurity structure maintained by :contentReference[oaicite:0]{index=0}. It is divided into 8 domains:

### 1. Security & Risk Management
- CIA triad (Confidentiality, Integrity, Availability)
- Risk management and compliance
- Ethics and governance

### 2. Asset Security
- Data classification and protection
- Secure handling and retention of information
- Data disposal processes

### 3. Security Architecture & Engineering
- Secure system design principles
- Encryption and cryptographic systems
- Security models like defense in depth

### 4. Communication & Network Security
- Network design and protection
- Secure communication channels (VPNs, firewalls)
- Protocol security

### 5. Identity & Access Management (IAM)
- Authentication and authorization
- Least privilege principle
- Access lifecycle control

### 6. Security Assessment & Testing
- Vulnerability assessments
- Penetration testing
- Security audits

### 7. Security Operations
- Incident detection and response
- SIEM and log monitoring
- Threat management and recovery

### 8. Software Development Security
- Secure coding practices
- Secure SDLC (Software Development Life Cycle)
- Application security testing

# Navigate Threats, Risks, and Vulnerabilities

## 🔐 Key Notes

- A **threat** is any potential event or actor that can cause harm to systems, data, or assets (e.g., malware, hackers, insider misuse).
- A **vulnerability** is a weakness in a system that can be exploited by a threat (e.g., weak passwords, unpatched software, misconfigurations).
- A **risk** is the likelihood that a threat will exploit a vulnerability and cause damage to an asset.
- Risk is influenced by:
  - Likelihood of a threat
  - Existing vulnerabilities
  - Impact of exploitation
- Common risk management strategies include:
  - **Acceptance** (accept the risk)
  - **Avoidance** (eliminate the risky activity)
  - **Mitigation** (reduce impact or likelihood)
  - **Transfer** (shift risk to third party)
