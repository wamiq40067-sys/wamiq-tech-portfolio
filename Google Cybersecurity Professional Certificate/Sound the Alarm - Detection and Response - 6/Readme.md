# Sound the Alarm: Detection and Response

This file contains my learning notes, key concepts, and understanding of cybersecurity incident detection, response processes, monitoring technologies, log analysis, and security operations concepts as part of the Google Cybersecurity Professional Certificate.

---

## Topic Overview

This course focuses on how cybersecurity professionals detect, investigate, and respond to security incidents in organizational environments. It introduces the incident response lifecycle, the responsibilities of security teams, and the importance of communication and documentation during incidents.

The course also explores detection and monitoring technologies such as IDS, IPS, EDR, SIEM, and SOAR tools. In addition, it covers packet analysis, log investigation, network traffic monitoring, and security event management used by security analysts and SOC teams to identify and respond to cyber threats.

## Key Topics Covered

### Incident Response Lifecycle

* Introduction to the NIST Incident Response Lifecycle
* Four main phases:

  * Preparation
  * Detection and Analysis
  * Containment, Eradication, and Recovery
  * Post-Incident Activity
* Importance of structured incident handling procedures

---

### Security Teams and Incident Response

* Roles and responsibilities of:

  * Computer Security Incident Response Teams (CSIRT)
  * Security Operations Centers (SOC)
* SOC analyst tiers:

  * Tier 1 (L1)
  * Tier 2 (L2)
  * Tier 3 (L3)
* Importance of communication, coordination, and escalation during incidents

---

### Incident Documentation and Playbooks

* Importance of maintaining accurate incident records
* Incident handler journals and chain of custody documentation
* Use of playbooks for standardized response procedures
* Documentation for investigations, compliance, and reporting

---

### Detection and Monitoring Tools

* Intrusion Detection Systems (IDS)
* Intrusion Prevention Systems (IPS)
* Endpoint Detection and Response (EDR)
* Security Information and Event Management (SIEM)
* Security Orchestration, Automation, and Response (SOAR)
* Real-time monitoring and alert generation

---

### Network Traffic and Packet Analysis

* Understanding network traffic flow
* Packet structure and packet headers
* Packet capture and analysis concepts
* Use of packet sniffing tools such as tcpdump
* Identifying suspicious network communications

---

### Log Analysis and Security Monitoring

* Importance of logs in investigations
* Types of logs:

  * System logs
  * Network logs
  * Authentication logs
  * Application logs
* Parsing and analyzing log data
* Correlation of events to identify threats

---

### SIEM and SOAR Platforms

* SIEM as a centralized platform for collecting and analyzing logs
* Aggregation, normalization, and correlation of event data
* Writing and running queries in SIEM tools
* Understanding automated response workflows with SOAR platforms

---

### Detection Signatures and Alerts

* Understanding how detection signatures work
* Components of signatures in IDS and SIEM tools
* True positives and false positives
* False negatives and true negatives
* Alert investigation and triage processes
