## Frameworks and Security Controls – Key Notes

Cybersecurity frameworks are structured guidelines that help organizations manage and reduce security risks in a consistent way. They define best practices for protecting systems, data, and operations.

Security controls are safeguards or countermeasures used to protect information systems from threats. These controls help prevent, detect, or respond to security incidents.

Controls can generally be grouped into:
- Preventive controls (stop attacks before they happen)
- Detective controls (identify and detect incidents)
- Corrective controls (recover and fix issues after an incident)


## Ethics in Cybersecurity – Key Notes

Ethics in cybersecurity refers to the moral principles and professional standards that guide how security professionals handle data, systems, and user privacy. It ensures that security practices are carried out responsibly and lawfully.

Cybersecurity professionals often have access to sensitive information, so ethical behavior is critical to maintaining trust and protecting individuals and organizations.

Common ethical principles include:
- Respecting user privacy and confidentiality
- Using data only for authorized purposes
- Avoiding unauthorized access to systems
- Reporting vulnerabilities responsibly instead of exploiting them
- Following laws, regulations, and organizational policies
