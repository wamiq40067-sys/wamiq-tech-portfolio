## Important Cybersecurity Tools – Key Notes

Cybersecurity tools are software and systems used to protect, monitor, and analyze networks, devices, and data. They help security professionals detect threats, investigate incidents, and prevent attacks.

Common types of tools include:
- Network monitoring tools to track traffic and detect unusual activity
- Vulnerability scanning tools to identify weaknesses in systems
- Intrusion detection and prevention systems to recognize and block attacks
- Log analysis tools to review system events and security records
- Encryption tools to protect sensitive data during storage and transmission


## Core Cybersecurity Knowledge and Skills – Key Notes

Core cybersecurity knowledge includes understanding how systems, networks, and data are protected from threats. It combines technical concepts, security principles, and real-world practices.

Key knowledge areas include:
- Networking fundamentals and how data moves across systems
- Security principles like confidentiality, integrity, and availability (CIA triad)
- Understanding different types of cyber threats and attack methods
- Awareness of operating systems and how they are secured
- Basic knowledge of risk management and security policies
