# Key Takeaways: Core Security Concepts

## 1. Safeguard Information

### Key Takeaways
- Protecting information is essential to maintain confidentiality, integrity, and availability (CIA triad).
- Security controls are used to reduce risks and prevent unauthorized access or data loss.
- Data must be protected in all states: at rest, in transit, and in use.
- Strong access control and monitoring help prevent data breaches and misuse.
- Regular backups and logging improve recovery and threat detection.
- Security is not a single tool but a combination of policies, processes, and technologies.

---

## 2. Encryption Methods

### Key Takeaways
- Encryption converts readable data (plaintext) into unreadable data (ciphertext) to protect it.
- It ensures data confidentiality, especially during storage and transmission.
- Symmetric encryption uses one shared key and is fast but requires secure key exchange.
- Asymmetric encryption uses a public/private key pair and is more secure for communication.
- Common algorithms include AES (symmetric) and RSA (asymmetric).
- Hashing is a one-way process used for data integrity, not encryption.
- Salting strengthens password hashing and protects against rainbow table attacks.
- Strong encryption depends on key length and secure algorithm design, not secrecy of the method.

---

## 3. Authentication, Authorization, and Accounting (AAA)

### Key Takeaways
- AAA is a core framework for managing user access and system security.
- Authentication verifies identity (who the user is).
- Authorization defines permissions (what the user can do).
- Accounting tracks user activity (what the user did).
- MFA and SSO improve authentication security and usability.
- Access control models include:
  - MAC (strict, centralized control)
  - DAC (data owner controls access)
  - RBAC (role-based access)
- Principle of Least Privilege reduces risk by limiting user access to only what is necessary.
- Separation of duties prevents fraud by dividing critical responsibilities among multiple users.
- Logging and auditing are essential for accountability and compliance.
