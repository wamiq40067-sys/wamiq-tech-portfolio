# Key Notes: Introduction to Networks, Network Communication, and Local & Wide Network Communication

# Introduction to Networks

## What is a Network?
- A network is a group of connected devices that can communicate and share resources.
- Devices in a network include computers, servers, smartphones, printers, and routers.
- Networks allow sharing of files, internet access, applications, and communication services.

## Purpose of Networks
- Share information and resources
- Enable communication between devices
- Support business operations and internet connectivity
- Improve collaboration and accessibility

## Common Network Components
- **Router:** Connects networks and forwards data
- **Switch:** Connects devices within a local network
- **Hub:** Broadcasts data to all connected devices
- **Access Point:** Provides wireless connectivity
- **Server:** Provides services and resources
- **Client:** Device that accesses network resources

## Types of Networks
- **LAN (Local Area Network):** Covers a small geographic area
- **WAN (Wide Area Network):** Covers large geographic areas
- **WLAN (Wireless LAN):** Wireless local network
- **VPN (Virtual Private Network):** Secure private communication over public networks

---

# Network Communication

## What is Network Communication?
- Process of exchanging data between devices over a network.
- Devices communicate using protocols and transmission media.

## Data Transmission
- Information is broken into smaller units called **packets**.
- Packets travel through different paths to reach the destination.
- Devices reassemble packets after receiving them.

## Communication Methods
- **Wired Communication**
  - Uses cables such as Ethernet or fiber optics
  - Faster and more stable
- **Wireless Communication**
  - Uses radio waves
  - Enables mobility and remote access

## Network Protocols
Protocols are rules that define how devices communicate.

### Common Protocols
- **TCP (Transmission Control Protocol):**
  - Reliable communication
  - Ensures packets are delivered correctly

- **IP (Internet Protocol):**
  - Provides addressing and routing

- **HTTP/HTTPS:**
  - Used for web communication

- **DNS (Domain Name System):**
  - Converts domain names into IP addresses

- **DHCP (Dynamic Host Configuration Protocol):**
  - Automatically assigns IP addresses

## IP Addresses
- Unique address assigned to a device on a network.
- Two main versions:
  - IPv4
  - IPv6

## Ports
- Logical communication endpoints used by services and applications.
- Example:
  - Port 80 → HTTP
  - Port 443 → HTTPS

# Local and Wide Network Communication

# Local Area Network (LAN)

## Definition
- A network that connects devices within a limited area such as:
  - Home
  - School
  - Office

## Features of LAN
- High speed communication
- Low latency
- Usually privately managed
- Easier to secure and maintain

## Devices Used in LAN
- Switches
- Routers
- Wireless access points
- Ethernet cables

## Advantages of LAN
- Fast file sharing
- Shared internet access
- Centralized resource management

---

# Wide Area Network (WAN)

## Definition
- A network that connects multiple LANs across large geographic areas.
- The internet is the largest example of a WAN.

## Features of WAN
- Covers cities, countries, or continents
- Uses public or leased communication lines
- Higher latency than LAN

## WAN Communication
- Data travels through multiple routers and networks.
- Internet Service Providers (ISPs) help connect WANs.

## Advantages of WAN
- Long-distance communication
- Remote access to resources
- Supports global business operations

---

# Differences Between LAN and WAN

| Feature | LAN | WAN |
|---|---|---|
| Coverage Area | Small | Large |
| Speed | Faster | Slower compared to LAN |
| Ownership | Private | Public/Private |
| Cost | Lower | Higher |
| Security | Easier to manage | More complex |

---

- Networks allow devices to communicate and share resources.
- Data communication happens through packets and protocols.
- LANs are used for small local communication.
- WANs connect networks across long distances.
- Protocols like TCP/IP are essential for communication.
- Routers and switches play major roles in networking.
