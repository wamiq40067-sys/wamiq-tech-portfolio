# Connect and Protect: Networks and Network Security

This file contains my learning notes, key concepts, and understanding of networking fundamentals, network security, protocols, and secure communication as part of the Google Cybersecurity Professional Certificate.

## Topic Overview

This course focused on how networks operate, how devices communicate, and how cybersecurity professionals secure network infrastructures against attacks. It introduced networking models, protocols, IP addressing, cloud networks, wireless security, and defensive security measures used to protect organizational systems and data.

## Skills and Concepts Learned

- Understanding computer networking fundamentals
- Difference between LAN, WAN, VPN, and cloud networks
- Basics of network architecture and communication
- Understanding TCP/IP and OSI networking models
- IP addressing and subnetting concepts
- Role of routers, switches, firewalls, and access points
- Network protocols and secure communication
- Wireless network security concepts
- Identifying common network attacks and threats
- Basics of network monitoring and defense strategies
- Understanding packet transmission and network traffic
- Awareness of security hardening and segmentation

## Key Topics Covered

### Networking Fundamentals
- What networks are and how devices communicate
- Types of networks (LAN, WAN, WLAN, VPN)
- Network topology basics
- Physical and wireless communication methods

### TCP/IP and OSI Models
- Layers of the TCP/IP model
- Layers of the OSI model
- How data moves across networks
- Encapsulation and decapsulation process

### Network Devices
- Routers and packet forwarding
- Switches and local communication
- Firewalls and traffic filtering
- Modems and access points
- Servers and client systems

### IP Addressing and Network Communication
- IPv4 and IPv6 basics
- Public vs private IP addresses
- MAC addresses and ARP
- DNS and DHCP functions
- Ports and network services

### Network Protocols
- HTTP and HTTPS
- TCP and UDP
- DNS, DHCP, FTP, SSH
- ICMP and network diagnostics
- Secure communication protocols

### Network Security
- Firewalls and intrusion prevention
- Virtual Private Networks (VPNs)
- Network segmentation
- Access control methods
- Wireless security standards
- Secure network configuration

### Common Network Threats
- Packet sniffing
- Spoofing attacks
- Denial-of-Service (DoS) attacks
- Man-in-the-middle attacks
- Malware spreading through networks
- Unauthorized access attempts
