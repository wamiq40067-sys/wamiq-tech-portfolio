useEffect Is a react hook which is used to run side Effects of components.
side effect = renderings in react connect with outside activity.
 		outside activities -->  FETCHING API
 					updation document state
 					setting timer (setTimeOut,setIntervals)

synxtax of useEffect -->

 			useEffect(() => {
 			   return() => {

 				}
 			},[]);


how useEffect works --> there are 3 main ways to use it, depending on the dependency array.

run on every render ->	useEffect(() => {

** 				console.log("Components rendered or updated")**
 			**});**

disadvantage->	can cause performance issue.

2. run only once -> If you pass and empty array(means empty dependency array) then it will run only once when the 		    component is mounting(creating component).

 				useEffect(() =>{
 					console.log("component rendered only once")
 				},[])

3. run on specific dependency change -> If you pass [variable] it will run when the variable changes.

 				useEffect(() =>{
 					console.log("count change");
 				}[count]);

eg :- useEffect(() =>{
 	console.log("will run when the name changes");
},[name]);

API (Application programming interface) --> Data Fetch
 	fetches the data from backend , and backend sends the request to database, and then sends back to backend 	and backend sends frontend

Things API needs to fetch - 1) UseState() -->(to store data)
 			  - 2) UseEffect() --> (when page renders, then API is called for that render)
 			  - 3) Fetch() --> (for API Calling)

 			UseEffect calls Fetch and then Fetch Calls the API and then API is stored in UseState


import {useState,useEffect} from react;

Function Student(){
 const[student,setStudent] = useState("");
 useEffect(() => {
 	//API Fetching
 	fetch("API url")
 	.then((response) => response.json())
 	.then((data) => {
 	setStudent(data);
 	console.log(data); //when we want to use console to show data
});
 },[]);
 	return(
 		<div> Student </div>

 		)
 	}
export Default Student;

fetching API with axios -(easy and powerful than fetch)
axios is a library which is used to call API . It is just like fetch but it is more powerful and easier.
In our real world mern projects , we will be using axios library more to fetch data. to use it we need install 
npm install axios

Difference between fetch and axios 
fetch -- we were converting data into JSON format (.then((res) => res.json()), complex and large code , manual error handling , for basic purpose 


axios -- automatic JSON format , code is clean and short , automatic error handling , for professionals and for industry 

basic Syntax --> 
			axios.get("API url")
			.then((response) => {
				console.log(response.data);
					})
			.catch((error) => {
				console.log("error");
					})
