synchronous - it means the code is running in a particular sequence of instructions given in the program. each instruction waits for the previous instruction to complete its execution.

asynchronous - sometimes important instructions get blocked due to some previous instructions, which causes a delay in the UI. asynchronous code execution allows to execute next instructions immediately and doesn't block the flow.
